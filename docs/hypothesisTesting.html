Hypothesis Testing

  • Home

What & why?

Used to check if an assumption about the population parameter is true or not.
Eg:
Research question: At what age does heart attack usually occur among the diabetes patients?
Claim: Age >70
Hypothesis testing checks how good the evidence supporting the claim is. At the end of this process, we can either reject the claim or fail to reject it.
So I can say that hypothesis testing is a formal proceedure used by statisticians to accept or reject a hypothesis/claim. Primarily used to assess claims made about the population parameter by experimenting on sample data taken from population.

Steps

There are two types of hypothesis for a given research question.

  • Null hypothesis (H0) :
    • Implies that sample observations result is purely by chance.
    • OR
    • The statement that there is no difference in the population.
    • Always has equality in it.(Eg: In the above example, Age=70 is H0)
  • Alternate hypothesis (Ha) :
    • Implies that sample observations result is influenced by some non-random pattern in it.
    • OR
    • The statement that there is some difference in the population.
    • Usually it is same as the claim
Proceedure
  • Writing hypotheses: State H0 & Ha. Both the hypothesis should be mutually exclusive.
  • Formulate analysis plans. Evaluation focused on which single test statistic.
  • Analyse sample data. Find value of test statistic and set significance level(alpha)
  • Interpret the results from above step and decide whether to reject or fail to reject H0.
How do we decide whether to reject H0 or not?
Based on the p-value and significance level.


Writing Hypothesis

While writing hypotheses, we need to know
  • Parameter that we are testing. Eg: Age in the heart attack problem.
  • Direction of the test(left,right,equality)
  • Value to be used in H0
How to know the direction of the test?

Condition Direction of test
Different from/Equal to Two tailed test
( Need to check the are a under curve on both the sides )
Eg: Age=80 or not.
Greater than Right tailed test( Need to check the area under standard normal curve that is greater than the H0 value. In other words, the area under the curve that lies to the right side of H0 value )
Less than Left tailed test( Need to check the area under standard normal curve where the values less than H0 value lie )


EXAMPLE:

Research Question Is the population mean different from μ 0 ? Is the population mean greater than μ 0 ? Is the population mean less than μ 0 ?
Null Hypothesis, H 0 μ = μ 0 μ = μ 0 μ = μ 0
Alternative Hypothesis, H a μ ≠ μ 0 μ > μ 0 μ < μ 0
Type of Hypothesis Test Two-tailed, non-directional Right-tailed, directional Left-tailed, directional

If we don't have good amount of data, then we can construct randomisation distribution under the assumption that H0 is true. We can use that distribution to find p-value and proceed further.

Calculating Test Statistic

  • In this step, we decide which test to use and which test statistic would be applicable in the given scenario.
  • If equality test, use double tailed test.
  • Is greater/less than test, use single tailed test.
  • We assume that the distribution of sample data is normal. Why?
  • Calculate the z-score of the given H0 value.
  • Look up the z-score value on z-table, to get p-value.
  • What is p-value you might ask!!! Read it here
  • In simple terms, when we plot the distribution of the sample values for the parameter we are trying to estimate, p-value is the probability that the observed results would be as extreme( or more like the ) as the test statistic (Test statistic is the value in H0)
  • To visualize it mathematically, it is the desired area under the distribution curve.

Decision Step

Check the p-value calculated in the above step.
See if it falls above the significance level (α)
If p>α -> fail to reject H0 -> Why?
If p≤α -> reject H0 -> conclude that the observed result didn't occur by random chance and that there is difference in the population.

Example

Research question:
Nancy read an article that said 26% of Indians can speak more than 2 foreign languages. She was curious if this figure was higher in her highly populated small town in India, so she tested
H0: p=0.26 vs Ha p >0.26,
where p = probability that one can speak more than 2 foregin languages. She found that 40 out of 120 people sampled could speak more than 2 foreign languages.

Solution:
Sample Prob = 40/120 = 0.333
z-score = (SampleProbability - H0Value)/StandardDeviation
= (0.333 - 0.26)/StandardDeviation
Calculating Standard Deviation:
SD = √(H0Val*(1-H0Val)/SampleSize) = √(0.26*0.74/120)
Substituting and calculating, we get z-score = 1.83
So we can say that z-score is 1.83 SD's away from the H0 value(mean in our case).
Values in z-score table usually give area to the left of the point we are looking for.
Since normal distribution curve is symmetrical we can check the z-score for -1.83 and that equals to 0.0336
So the p-value for our test = 0.0336.
Usually the significance level(α) is taken to be 5% (0.05)
Since the p-val≤α, we reject H0.

Errors in HT

Two possible decisions can be made in a hypothesis test.
1. Reject H0
2. Fail to reject H0

Hypothesis testing uses data from a sample to estimate an unknown population parameter.
As a result, the inference may or may not be true.

We could commit errors in these 2 scenarios.
1. Falsely rejecting H0, while H0 is actually true.(TYPE-I error)
α = Probability(Type-I error)

2. Failing to reject H0, while H0 is actually false.(TYPE-II error)
ß = Probability(Type=II error)

Decision Reality
H0 is true H0 is false
Reject H0, (conclude Ha) Type I error Correct decision
Fail to reject H0 Correct decision Type II error


TYPE-I vs TYPE-II

When is higher α level preferred?
Higher α -> Higher area that opposes H0 at both the tails of normal curve. -> Chance of type-I error decreases -> Chance of Type-II error increases.



Note: α is not a value in distribution, whereas it is the level. Like α=0.05 means that towards the rear on both the tails, there lies 5% of the data.

Pilot studies (smaller studies performed before a larger study) often use a higher level because their purpose is to gain information about the data that may be collected in a larger study; pilot studies are not typically used to make important decisions.

When is Type-1 err more dangerous than a Type-2 error?
In medical reasearch, we don't want to give a False Positive result to the patient, which makes the patient feel the treatments being ineffective. So in such cases, Type-I error is more dangerous than Type-II error.
So we set smaller α values, so that even smaller probabilities supporting H0 can make it fail to reject H0. This makes it more difficult to rehect H0 and reduces probability of committing a Type-I error.

Note: α level of 0.05 means we are accepting 5% chance of making a Type-I error.
