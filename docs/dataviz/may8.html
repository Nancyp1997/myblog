Sets & Types

Sets are subset of your data linked to a condition. There are 3 types of sets.
  • Constant sets[Multi dimensional]
  • Computed sets[One dimensional]
  • Combined sets[Multi dimensional]
Not sure what dimension is!! Click here!

CONSTANT SET
  • Constant sets are multi dimensional i.e, const sets can have more than one column in its data.
  • These have to be created manually.
  • Check this video to see how a constant set is created.
  • Created constant sets are placed under dimensions (eventhough they have continuous data in it because after being placed in set, the values are countable).

How do I check the items in the constant set?

Note: Distinct features of constant set is that even if the underlying data changes, the data in constant set doesn't change.
To get the data of constant set refreshed or to get new updated data which is being overridden due to constant set creation, we can remove the data from the set.
COMPUTED SET
  • Uses some underlying condition logic to create the set.
  • Unlike the constant set, computed set dynamically updates its members on auto-refresh, if there are any underlying changes in the data.
  • It is one dimensional i.e, you can have only entries of only one column in the computed set.
  • How to create it?
    • Hover over to the dimension of choice
    • Click the drop-down icon.
    • Create set
    • While creating set, set the condition like 'Top3' or 'bottom 3' or 'middle 5'
COMBINED SET
  • Select any set.
  • Click on drop down-> Create combined set.
  • In the dialog box, select the other set you want to combine it with.
  • CAN BE CREATED FOR BOTH ONE DIMENSIONAL AND MULTI DIMENSIONAL.
  • Try creating two sets - top 2 by some condition and bottom 2 by same condition.Create combined set. Eg: Top and bottom 2 item-categories by sales.
Note: Only two sets with same dimensions can be combined with one another.

Bins

Bins are created to analyze the distribution of any quantitative variable. They are containers of equal size that hold data that fall in its range.
We use bins in bar charts, histograms.
In bar chart --> Gaps between bins are present.
In histogram --> No gaps are present between bins or very thin gaps indicating continuous in nature.
In general, we need to visualize the distribution of a continuous variable to check if it follows normal distribution or not.
For that, in Tableau, we can either use histogram or create bar chart We can visualize the distribution of a quantitative variable using either "bin vs cnt(Column)" or by plotting "histogram" directly on quantitative column.

BINS - BAR CHART
Steps:
  • Click on drop down of measure(quantitative variable)
  • Create bin
  • We can choose bin size or choose to keep the default bin size.
  • Tableau creates a default bin size by some formula.We can either continue or choose another bin size.
  • In the popup, the value under 'cntD' indiciates the number of distinct values for a column.
  • Generates a count of values that lie in the bin range and generates the height of the bar.
After creating bins on 'measures'(quantitative variable), the bin now goes under dimensions(categorical variable) becuase now we are assignning a count to each distinct value of the measure.
This is called "binned dimension".
HISTOGRAM
We plot the histogram directly on the continuous quantitative variable. So, the gaps between bins in a histogram is very thin.
  • Drag-drop the measure to rows area
  • In top-right corner, click on Show Me, select Histogram graph option.

Parameters

Parameters are used to make the visualisation more dynamic or interactive by letting users choose the values to be displayed.
Parameters are generally used to get input from the end user, giving more power to the user than the author in setting values of the viz. It then applies some logic using dynamic values, taken from the user and displays the worksheet accordingly.
They are dynamic values that replace the constant values in a viz.
Get input form user -> Apply some formula -> Changes displayed in worksheet accordingly -> It can replace any value with dynamic values.
Filters are used to narrow down the data.

There are three types of parameters.

Parameters with filters & sets
For a measure we cannot add a parameter to its filter.
For a dimension:
Create a new parameter by clicking on icon on top right corner of left menu.
  • Edit filter of any dimension
  • Go to Top tab -> By field row
  • Top val -> Click on create new parameter or use already created parameter there or enter a value manually.
  • Parameters are displayed separately at the bottom of left menu.
  • Click on the parameter, which we have used in step 3, drop down and select 'Show Parameter'.
  • This displays the slider on top right.
  • Move slider and now we can dynamically change values.
Eg: For sample superstore dataset, plot Sales vs State, in row area, click on State -> Create filter -> Top tab -> Create parameter ->show it in viz

Parameters in Sets
In sets, we can create a parameter only in the 'Top tab' for computed sets only.
Eg:
Create a computed set on any dimension(State) by clicking on its drop down on left menu. While creating the set, go to Top tab, create a new parameter there for the Top tab value.

Parameters with bins
  • For any measure, we can create bins for it.
  • While creating bins, we need to choose the bin size.
  • We can make user choose the bin size by creating a parameter for it.
  • When we create a histogram for that measure or use the binned dimension, we can enable the 'Show Parameter' on the viz.
  • When u create a param through the measure dialog box, (In sales, when creating bins,if u click on new param, ), we can only choose data type of that particular measure.
Eg:
For sales measure, create bins, in top tab , create a new parameter.
Parameters with calculations in Tableau
  • Create a parameter with corresponding data type.
  • Create calculated field
  • Drag drop calculated field to color in graph
  • Calculated field comes under dimensions
  • How to make calculated field dynamic? Using 'PARAMETER'!!!!
  • Replace the static value with the parameter created in step 1
Eg: States having sales less than 1 lakh sales are low income, rest are high income states.
1. Create a parameter to set the border that distinguishes high sales from low sales states. Int data type and some value.
2. Plot Sum(Sales) vs State
3. Create a calculation field on Sales specifying the condition and using the parameter created in Step - 1
4. Drag drop the calculation onto the color of the graph.
5. Show parameter and alter it in viz

In filters and sets -> only in top tab, a param can be used.
In bins, only in bin sizes, a param can be used.


Actions

What is action in Tableau & wts its purpose?
Parameters can make values in viz dynamic and interactive. Actions can make viz even more user interactive.
Users can interact with your visualisations by selecting marks, or hovering, or clicking a menu, and the actions you set up can respond with navigation and changes in the view.
There are six types of actions.
  • By filter
  • By highlight
  • By Go to sheet
  • By URL
  • Change set values
  • Change parameter

Sheet 1: Category vs sum(sales) Sheet 2: Sub category + Category vs Sum(Sales) Create new dashboard -> Drag drop both sheets to the dashboard. Usually 5 to 6 sheets are placed in a dashboard. How to create it? Dashboard menu -> Actions Each one in detail below:
Filter Actions
  • Filter actions pass information between worksheets or sheets inside a dashboard.
  • We can either select/hover/rightclick-menu-actionfilter on source sheet/s and the action would be performed on the target sheet.
  • We can also choose what happens in the target when we clear our selection in source sheet. (Leave all values/Show all values/Exclude all values)
  • How to create it?
  • Steps:
    • Worksheet/Dashboard -> Actions
    • Add Action->Filter
    • Select source sheet/s and target sheet
    • Select actions (Select/Hover/Menu)
    • Select what to happen after clearing the filter

Hlighlight actions
Using highlight actions, we can highlight the marks of interest by brightening the area and dimming the other areas. Steps are same like creating filter action.
Go to URL action
  • Click on an area, it gets redirected to a webpage from Tableau.
  • You need multiple search values so to separate each item in search query u specify item delimiter
  • Eg: If item delimiter is ',' and u select it1 and it2 in viz, then URL would be "baseURL=it1,it2"
  • If items selected from viz have any special characters in them, we can replace them with specifying some value in 'Delimiter Escape field'
  • Eg: If we have an item like '#petbird' and delimiter escape as ',' then the item in URL gets replaced as ',petbird'
Parameter Actions
Upon changing a parameter in a source sheet, the targeted values/plots in the targeted sheet should get updated.
  • Create a parameter on Category 'param'.
  • Create a calculated field 'cf1' using the above created parameter.
  • IF [Category] ==[param] THEN [Subcategory] ELSE "" END
  • Plot 'Category + Subcategory' vs 'Sales(Sum)'
  • Show the parameter slider on viz.
  • Plot Category + cf1 vs SUM(Sales)
  • Click on any category in plot or select the category from the parameter dropdown.
  • We can see the data values changing
