Boxplot



  • Used mainly to identify outliers.
  • Used to identify median, Q1,Q3.
  • InterQuartile Range (IQR)= Q3-Q1
  • To detect an outlier we use IQR. Any point p is an outlier if p<(Q1- 1.5*IQR) or p>(Q3 + 1.5*IQR)

Steps:
  • Plot measure vs any other dimension.
  • If the measure has any aggregation, then remove it. How?
  • In the "Show Me" pane on RHS, select box plot graph.

Dashboards and story points

Plot following
Sheet-1: Category+Sales with Color:Region.
Sheet-2: Cat+sub-cat vs Sales with Region on Color.
Sheet-3: Category+Latitude vs longitude with Color:Region, Detail: state
Create a new dashboard with above created 3 sheets.
Click on Size=>Automatic to make dashboard fill the entire screen width.
Funnel icon on sheet present in dashboard -> Shortcut to action by filter->the sheet which has the icon is source sheet and rest are target sheets.
This can be activated on any sheet in dashboard making that the source sheet and the remaining the target sheets.
If we want any sheet to be excluded from being changed upon actions in any other sheet in dashboard, then click on the sheet(which we dont want to change) dropdown -> Ignore Actions.This sheet will be ignoring actions applied on any other action on dashboard.
To create an interactive filter on dashboard, click on dropdown of any sheet (which has same sheet as target) -> Filters -> Column name.
To convert this to a global filter(entire dashbaird to be affected including source sheet), click on dropdown of interactive filter->apply to worksheets->all using this data source.
We can add caption to every worksheet by clicking on dropdown for each sheet on dashbboard- Usually it has default description of the worksheet.
By default auto updates are enabled. Inorder to improvise performance, incase of very large data, we can disable 'auto updated'.
Action by highlight-> Sheet dropdown-> Highlight-> col name
If we want a floating sheet in dashboard ( without it filling entire space), click on dropdown of that sheet, make it floating.

Objects in dashboard
Can be found on the left pane of Tableau dashboard.
When we have few horizontal worksheets in a dashboard, whose height doesn’t fit the screen length (in such cases, a scroll bar appears avoiding graph cutoffs).
To automatically align them horizontally or vertically to fit entire height, we can use containers in dashboard that automatically fits each of the sheets according to their alignment. To do this, drag-drop one horizontal and one vertical containers to the dashboard before dragging any sheets to the dashboard.
Can also add a webpage object by drag-drop to the dashboard from the Objects pane in left of dashboard.
Blank space object is used to give enough space between the objects in dashboard (used for aesthetic purposes), whose width/height can be manually adjusted.

Storypoint
Storypoints= arrangement of worksheets + dashboards arranged in a sequential way as if narrating a story.
In first page on stnrypoint, add a sheet/dashboard then add blank page and in next page, add another sheet/dashboard and so on.


Rank

It’s a quick table calculation and we have 5 different functions under Rank.
Based on only measures, we will be able to rank the data.
If want to arrange data based on the highest value of a column having Rank -1 then use Rank calculation
Create a new calculationin worksheet.-> RANK(SUM(SALES))
Simple rank- same position more than 1 means the next n ranks get omitted.
Rank dense- no omission for any rank
Rank modifed= omits previous lower rank
Unique rank- each row gets unique rank based on the aplhabetical order of dimension incase of same measure values.
Rank percentile - percentile based on the measure.
